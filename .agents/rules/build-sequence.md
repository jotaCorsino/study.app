# build sequence — studyhub

## official order
1. create the MAUI Blazor Hybrid solution and project structure
2. create shell, navigation, and core layout
3. create home/catalog page with mock courses
4. create internal course page with mock modules, topics, and lessons
5. create dashboard/progress area with mock values
6. create roadmap page with mock roadmap data
7. create complementary materials page with mock curated content
8. only after approval: implement local persistence with SQLite
9. only after approval: implement real folder import and indexing
10. only after approval: implement real local video playback workflow
11. only after approval: implement AI and YouTube integrations

## principle
Never jump into backend or integrations before the frontend experience is coherent, reviewable, and approved.
